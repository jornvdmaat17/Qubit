Maak hier een JSON file om uit te voeren

  • Aparte Qubits moeten zo ingevoerd worden: [0,1],[1,0], enz
  • Wanneer [] wordt ingevuld wordt er een random Qubit aangemaakt.
  • Meerdere indices kunnen worden ingevoerd als 0,1, enz
  • Maak een complexe qubit door door het zo in te vullen: ["1j", "0j"], let op de dubbele quotes en de 1 en 0 voor de j.
  • Er is ook een mogelijkheid om een half wortel 2 in vullen, maak je qubit dan zo: ["HS2","HS2"], let weer op de dubbele quotes.
  • Als je de tussenstappen wil printen voer bij print dan true of false in, zonder quotes!
